BELLE'S BEAUTY & CO. — Asset Files
====================================

Place the following image files in this folder (assets/):

  logo.png        — The Belle's Beauty & Co. circular logo
                    (the image you uploaded with the floral wreath and script text)

  hair-makeup.png — The hair & makeup line-art illustration
                    (brush, curling iron, lipstick, compact mirror, floral details)

  nails.png       — The nails line-art illustration
                    (hand with long nails, nail polish bottles, nail tools, florals)

Notes:
- PNG format is recommended to preserve transparency/quality.
- If your files are .jpg, rename them to .png OR update the src= paths
  in index.html to match (e.g. src="assets/hair-makeup.jpg").
- If the image files are missing, the site will display built-in SVG
  illustrations automatically as a fallback — so the site always looks good.
